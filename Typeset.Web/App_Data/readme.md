Basic Structure
---
A basic Typeset site looks something like this:

    App_Data
    |-- _includes
    |   |-- footer.html
    |   `-- header.html
    |-- _layouts
    |   |-- home.yml
    |   `-- post.yml
    |-- _posts
    |   |-- 2012-02-17-stay-classy-san-diego.textile
    |   `-- 2012-07-05-typeset-a-lightweight-markup-blog-engine.md
    |-- images
    |   `-- avatar.png
    |-- scripts
    |   `-- javascript
    |       `--jquery.min.js
    |-- styles
    |   `-- css
    |       `--reset.min.css
    |-- _config.yml
	|-- about.md
    `-- other.html
